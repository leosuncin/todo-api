## Description

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn dev

# debug mode
$ yarn debug

# production mode
$ yarn start
```

## Test

```bash
# run tests
$ yarn test

# watch tests
$ yarn test:watch

# test coverage
$ yarn test:cov
```
